Sobre mí

Soy Henry Steeven Jaimes Bastos, estudiante del programa ADSO (Análisis y Desarrollo de Sistemas de Información) en el SENA. Este proyecto corresponde a la actividad de maquetación y consumo de APIs, implementado con tecnologías web básicas: HTML, CSS y JavaScript (Vanilla), y construido con Vite para el flujo de desarrollo.

El backend se desarrolló con Supabase y ofrece una estructura sencilla de tablas relacionadas para categorías, productos y pedidos, facilitando la carga y gestión del catálogo. El propósito de esta entrega es demostrar buenas prácticas de maquetación, organización de código y consumo de un backend ligero.

Tecnologías: HTML · CSS · JavaScript (Vanilla) · Vite · Supabase
